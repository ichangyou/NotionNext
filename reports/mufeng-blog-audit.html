Schema (JSON-LD) Fail
· No JSON-LD schema found in page source. · Homepage should declare WebSite + Organization at minimum. · Missing WebSite schema means Google cannot generate sitelinks search box.

Also add Organization with logo and sameAs for social profiles.

Top 3 Priority Actions

1
Rewrite Title & Meta Description

Current title (9 chars) and description (17 chars) are unusable for ranking. These are the two most direct SEO levers on any page and require only a config change in blog.config.js or the Notion page properties.

2
Fix Soft 404 — Return Proper HTTP 404 Status

Every non-existent URL currently returns 200. Google may crawl and index garbage URLs, wasting crawl budget and polluting the index. In Next.js, add notFound: true to catch-all routes or use notFound() in the App Router.

3
Add JSON-LD Schema to Homepage

Zero structured data means no rich results eligibility and no sitelinks search box. A WebSite + Person schema block in <Head> takes ~10 lines and is immediately crawlable since it goes into static HTML.
